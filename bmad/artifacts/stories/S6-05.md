# S6-05 — Cross-platform build validation (Electron, Android)

## Goal
Verify that the application builds and runs correctly on Electron (Desktop) and Android (Mobile).

## Tasks
**Electron:**
- Run `npm run dev:electron` and verify:
  - App launches without errors
  - Chat UI renders correctly
  - Send/receive messages works
  - Settings page accessible
- Run `npm run package` (or electron-builder) and verify:
  - Build completes without errors
  - Generated executable runs on Windows

**Android:**
- Run `npm run build` then `npx cap sync android`
- Open Android Studio: `npx cap open android`
- Build APK and install on emulator/device
- Verify:
  - App launches
  - Chat UI is responsive
  - Send/receive messages works
  - Deep linking works (if configured)

## Acceptance
- Electron build runs on Windows without errors
- Android APK installs and runs on device/emulator
- No console errors specific to either platform
- Document any platform-specific issues found
