S3-01 — SkillAutocomplete component

Goal
- Implement client component SkillAutocomplete.svelte for searching and selecting skills/slash commands.

Tasks
- Create component in client/src/lib/components/SkillAutocomplete.svelte (Svelte 5 runes)
- Fetch suggestions from GET /api/skills?query=
- Debounce input (200ms), keyboard navigation, accessible ARIA roles
- Emit selection event with full skill object
- Unit tests (vitest) and basic story in storybook/draft

Acceptance
- Suggestions shown after 3 characters
- Arrow keys navigate, Enter selects, component emits selection
- Tests cover debounce and keyboard selection
