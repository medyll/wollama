# S7-03 — Android build verification

## Goal
Verify Android APK builds and runs correctly on device/emulator.

## Tasks
**Build:**
- [ ] Run `npm run build` — client builds successfully
- [ ] Run `npx cap sync android` — syncs to Android project
- [ ] Open Android Studio: `npx cap open android`
- [ ] Build APK: Build → Build Bundle(s) / APK(s) → Build APK(s)

**Testing:**
- [ ] Install APK on physical device (Android 10+)
- [ ] Install APK on emulator (Android 12+)
- [ ] Test app launch
- [ ] Test chat: send/receive messages
- [ ] Test settings: open, toggle skill/hook
- [ ] Test deep linking (if configured)
- [ ] Test microphone permission for voice input
- [ ] Test responsive layout on phone screen

**Performance:**
- [ ] App launches in < 3 seconds
- [ ] Messages scroll smoothly (60fps)
- [ ] No memory leaks (check Android Studio profiler)

## Acceptance
- APK builds without errors
- Installs and runs on device/emulator
- Core features work on mobile
- No critical crashes or ANRs
