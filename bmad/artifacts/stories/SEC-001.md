# SEC-001 — Update TLS configuration and add security headers

Module: infrastructure / security
Severity: Critical
Owner: Security Lead
Target Date: 2026-03-28

Issue:
TLS configuration is outdated (TLS 1.2) and security headers (HSTS, CSP) are missing.

Evidence:

- Server reports TLS 1.2
- No HSTS, missing CSP and other recommended headers

Remediation:

- Enforce TLS 1.3 across servers and proxies.
- Add and verify security headers: HSTS, CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy.
- Update deployment configuration and test in staging.

Acceptance Criteria:

- TLS 1.3 is enforced end-to-end.
- HSTS and recommended security headers are present and validated by automated checks.
- No breaking changes for clients.
