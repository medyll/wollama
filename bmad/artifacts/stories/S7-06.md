# S7-06 — Accessibility audit (RGAA 2.0)

## Goal
Ensure the application meets RGAA 2.0 accessibility standards.

## Tasks
**Audit:**
- [ ] Run automated accessibility audit (axe-core, Lighthouse)
- [ ] Manual keyboard navigation test
- [ ] Screen reader test (NVDA on Windows, VoiceOver on macOS)
- [ ] Color contrast audit

**Fixes:**
- [ ] Add missing ARIA labels to all interactive elements
- [ ] Ensure all images have alt text
- [ ] Fix color contrast issues (text on backgrounds)
- [ ] Ensure focus indicators are visible
- [ ] Add skip-to-content link
- [ ] Ensure all forms have proper labels

**Testing:**
- [ ] Navigate entire app with keyboard only
- [ ] Test with screen reader enabled
- [ ] Test with high contrast mode
- [ ] Test with zoom (200%)

**Documentation:**
- [ ] Create accessibility statement
- [ ] Document known accessibility issues
- [ ] Add accessibility guidelines for future development

## Acceptance
- All critical accessibility issues fixed
- Keyboard navigation works throughout
- Screen reader can navigate all pages
- Color contrast passes WCAG AA
- Accessibility statement published
