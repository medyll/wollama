S3-02 — /api/skills invocation pipeline

Goal
- Implement server-side pipeline for executing skills and providing autocomplete data.

Tasks
- Add routes in server/src/routes/skills.ts: GET /api/skills, POST /api/skills/execute
- Implement middleware chain for pre/post hooks (server/hooks/*)
- Define shared types in shared/ for SkillDefinition and InvocationPayload
- Unit tests for middleware behavior and routing
- Integrate a mock skill handler for initial end-to-end test

Acceptance
- GET returns matching skills for query
- POST goes through hooks and returns standardized result
- Tests validate hook short-circuiting and payload transforms
