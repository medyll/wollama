## Problème

## Cas à couvrir

- Expiration de token
- Flux de refresh token
- Tokens invalides / malformés
- Concurrence (double-login, race condition)

- [ ] Tous les tests passent en CI
- [ ] Tests ajoutés sous `tests/auth/` avec nommage explicite
- [ ] Seuil de couverture enforced dans la config CI
